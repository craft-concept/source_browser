## Installation
Add this line to your application's Gemfile:

```ruby
gem "source_browser"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install source_browser
```


Mount the engine in your `config/routes.rb`
```rb
mount SourceBrowser::Engine, at: "/source"
```
